ARM PB1176 Configuration for DS-5

1.- Set the DAP enable jumper in the board

2.- Create a new debug configuration in DS-5:

    - Connection->Platform: ARM - PB1176JZF-S
    - Connection->Project type: Bare Metal Debug
    - Connection->Debug Operation: Choose the one you prefer
    - Files->Application: safeg.elf
    - Debugger->execute_commands:
    
        loadfile Z:\kernels\safeg-toppers-svn\trunk\safeg\trust.elf
        loadfile Z:\kernels\safeg-toppers-svn\trunk\safeg\non-trust.elf
